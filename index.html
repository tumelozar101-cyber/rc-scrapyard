How RC Scrapyard Can Use This

This section is designed for people who want to sell used, non-running or accident-damaged vehicles. Photos and basic details help RC Scrapyard decide quickly if they’re interested.

  • Receive clear info: make, model, year, location and condition.
  • View photos before committing to a call-out.
  • Keep a record of offers and vehicles viewed.

We can customise this process – for example, sending details straight to a WhatsApp number, an email inbox, or into a simple internal dashboard.

How This Can Work for RC Scrapyard

This page is a starting point. From here we can build a full system around how you run the yard.

The goal is simple: make it easier for customers to find the parts they need, and make it easier for RC Scrapyard to handle enquiries and manage stock.

We can build this in phases, for example:

  • Phase 1 – Online Enquiries: Customers search parts and send requests to RC Scrapyard.
  • Phase 2 – Stock Management: Private login for staff to add vehicles and parts, mark items as sold or reserved.
  • Phase 3 – Extras: Images of parts, automatic WhatsApp/email replies, exportable reports of requests, etc.

Every colour, button and section here can be changed. This is simply a prototype so you can see how RC Scrapyard could look and work online before we commit to final features.
